# mabyko

Korea-born, globally built.

mabyko is an engineering team building practical tools for developers and AI agents.  
We focus on workflows that make software work clearer, safer, and easier to repeat.

## What We Build

- Agent skills and reusable workflows
- Developer productivity tools
- Automation for coding, documentation, review, and release work
- Open-source utilities shaped by real engineering practice

## Principles

- Build useful things first
- Keep workflows explicit and repeatable
- Prefer small, composable tools
- Document decisions clearly
- Design for global developers

## Open Source

Our projects are built in public where possible.  
We welcome issues, ideas, and contributions from developers everywhere.

## Language

We work in English by default for global collaboration.  
Some projects may also include Korean documentation.

## About the Name

`mabyko` comes from “made by koreans.”  
It reflects where we started, not a boundary around who can join or use what we build.
